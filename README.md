# Sign-Sacker (签名掠夺者) - 2.0

## introduce

Sign-Sacker(签名掠夺者)：一款数字签名复制器，可将其他官方exe中数字签名，图标，详细信息复制到没有签名的exe中，作为免杀，权限维持，伪装的一种小手段。


## 更新 2.0

分别导入含有签名和需要签名的文件，点击生成即可，玩免杀的可能会用到。

新增功能：

1. 掠夺高清图标，完美复刻。
2. 掠夺exe详细信息，以假乱真。
